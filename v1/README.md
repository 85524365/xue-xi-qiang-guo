# 基于 Autojs 的《学习强国》自动学习脚本v1

1. 只刷阅读文章、视听学习。
2. 只是模拟点击，封号应该不会（理论上哈，被封别找我！）。
3. 只限制 android 系统。
4. 仅用于技术学习于交流，不可用于作弊。

## 实测

### 设备

* MI Mix2 (1080*2160)
* Autojs 4.0.1Beta
* 学习强国 2.11.1

### 效果

* 主动进行视频学习 18-21 分钟
* 主动进行文章学习 12 分钟以上

### 已知 Bug

本质是基于位置的点击，当点击的文章是视频时会错乱，这时候只能手动进入 Autojs 停止脚本再运行了。
